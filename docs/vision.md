### Назва проєкту: TextEncoder.

### Короткий опис: 
    Консольний застосунок для миттєвої конвертації звичайного тексту в спеціалізовані формати (азбука Морзе, шрифт Брайля) зі збереженням історії сеансів перекладу.

### Цільові користувачі: 
    Люди, які вивчають шифри чи системи комунікації; розробники, яким потрібен інструмент для генерації тестових даних у цих форматах.

### 3 основні сценарії використання:

    Конвертація введеного тексту в обраний формат (Морзе/Брайль).

    Перегляд історії останніх конвертацій у поточній сесії.

    Очищення історії перекладів.

### 3 нефункціональні вимоги:

    Простота розширення: додавання нового шифру не повинно вимагати зміни існуючого коду (Open/Closed Principle).

    Відновлення після помилки: застосунок не повинен падати при введенні непідтримуваних символів.

    Тестованість: бізнес-логіка (переклад) має бути ізольована від вводу/виводу для легкого покриття unit-тестами.

    Обмеження першої ітерації (Lab 34): Дані зберігаються лише в оперативній пам'яті (In-Memory). Зворотна конвертація (декодування) свідомо не реалізується. Немає графічного інтерфейсу.
